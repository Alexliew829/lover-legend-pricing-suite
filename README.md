# Lover Legend Combined Calculators V1.0

这个 GitHub Pages 项目把两个原有系统放在同一个页面：

- `cost-calculator/`：成本与售价计算器
- `bonsai-price-calculator/`：盆景价格计算器 V2.2

两个系统的原有计算逻辑、HTML、CSS 和 JavaScript 均未修改；根目录 `index.html` 只负责组合显示。

## 发布到 GitHub Pages

1. 建立新的 GitHub repository。
2. 上传本压缩包解压后的全部内容，确保根目录直接看到 `index.html`。
3. 在 **Settings → Pages** 选择：
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /(root)
4. 保存后等待 GitHub Pages 发布。

电脑版会左右并排；较窄屏幕及手机会自动改为上下排列。
